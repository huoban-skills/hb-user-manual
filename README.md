# hb-user-manual

伙伴云应用模块使用手册生成 Skill。

这个 Skill 用于根据伙伴云应用的真实表结构、字段、关联关系和自动化配置，生成面向业务人员的模块使用手册。

## 适用场景

- 使用手册
- 操作手册
- 模块使用说明
- ERP 手册
- 给业务人员看的系统操作指南
- 将伙伴云应用模块整理成可发布的 Markdown 手册

当用户提到“使用手册”“操作手册”“模块使用说明”“帮我写这个模块怎么用”“这个模块的使用指南”“ERP手册”等场景时，优先使用本 Skill。

## 工作流程

本 Skill 按四个阶段执行：

1. 确定范围：先获取表清单，让用户确认纳入手册的表和顺序。
2. 批量采集元数据：采集字段结构、关联关系和自动化配置。
3. 按模板生成文档：依据写作规范生成面向业务人员的 Markdown 手册。
4. 用户确认后同步飞书：用户确认本地 Markdown 后，再创建或更新飞书文档。

## 核心原则

- 数据驱动，不靠猜。
- 关键节点等用户确认。
- 面向业务人员写作，不写技术文档。
- 字段结构、关联关系、自动化规则优先通过 CLI 或 API 采集。

## 安装

将本仓库克隆或同步到 Claude Skills 目录：

```bash
git clone https://github.com/huoban-skills/hb-user-manual.git ~/.claude/skills/hb-user-manual
```

如果目录已存在，可先确认本地没有未保存修改后再更新。

## 文件

- `SKILL.md`：Skill 主说明文件，包含触发规则、流程和执行约束。
- `references/writing-guide.md`：使用手册写作规范。
- `scripts/collect_meta.py`：元数据采集脚本。
